--- title: Mask ---
> To use a mask you must declare the `data-av-mask` attribute and assign it a type.
> Currently the defaults types are `PHONE`, `DATE`, and `SSN`.
> You can roll a type you simply write it out, `9` being for digit, `a` for letter, and `*` for either. > The attribute `aa9a*` can accept "ab1c1" or "ab1cd"
