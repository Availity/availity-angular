> For better performance, query for all permissions that your app will need on load.
> The following gives example of how to use the `av-has-permission` directive to hide a section based on whether or not user has permission. Or in last link will send to system unauthorized page.

  • Not Authorized to 999
  • Is Authorized to either 452 or 999
  • Click to change from 452 to 999
  • Click to change from 452 to 999, go to error page
